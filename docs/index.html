Estimating carbon from lidar

Last update: 01-Jun-2021


Hi! I created this website to keep you updated with the project! First, I’ll give an overview of what we want to do and then the different processes and analysis I have done so far. Main tabs of this website explain further how I processed data for each of the elements of the model (i.e., AGB, TCH, BA & ρ) as well as the modelling process.


Project overview

The main goal is to generate equations to estimate aboveground biomass/carbon (AGB/ACD) from lidar. We want to better understand variation in Wood Density (ρ) and Basal Area (BA) to improve the Top-of-the-Canopy Height (TCH) - Biomass relationships. The aim is to generate the following model:

\[ ACD_{plot} = a · TCH^d · BA^c · ρ^b \]

ACD will be calculated using field plot data by summing above-ground biomass (AGB) for each tree (Chave et al., 2014) and then multiplying by 0.47, which is the conversion factor for carbon content (Martin & Thomas, 2011).

LiDAR data will be used to calculate TCH, while BA and ρ (aka WD) are derived from field plot data. Our aim is to provide a better understanding on how BA and ρ affect ACD-TCH relationships to improve carbon estimations.

Data collection

While other datasets are yet to come, I am working with forest inventories plus matching lidar from Paisagens Sustentáveis (PaiSus) and Marcus Vinicio Neves d’Oliveira (MarVin) in Brazil.

Cleaning inventory data

Inventory datasets were compiled and standardised from collaborators shown above. Taxonomic errors were corrected using the Taxonomic Name Resolution Service v5.0 (https://tnrs.biendata.org/).

All inventories matching with lidar (743plots/tansects in 26 locations) were selected, but some data were excluded in the following sequence:

  • 112 inventory transects (i.e., 500 meter transects in Paisagen Sustentáveis).

  • 177 re-sampled plots. I selected most recent (not always) with the least time difference from its lidar survey.

  • 8 plots which had more than half of the trees dead and less than 10 individuals.

  • 3590 dead individuals were removed. (I may add them later into biomass estimations, 3377 dead trees (94%) have DBH measurements). NOTE: MarVin dead trees are…

  • 122 lianas.

  • 19 trees without DBH measurement.


Finally, what’s left for the analysis are 65039 individuals in 446 plots in 24 locations:
